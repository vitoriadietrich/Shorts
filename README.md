# Loading Animation HTML CSS #shorts

Um projeto simples de **animação de carregamento (loading)** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar uma **loading animation rápida e moderna**, ideal para vídeos curtos (#shorts) e interfaces web.

## 🛠️ Tecnologias

- HTML
- CSS (animation, keyframes)

## 🎯 Funcionalidades

- Animação de carregamento
- Design simples e leve
- Fácil de reutilizar

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
